== "A curated list of awesome Model Context Protocol (MCP) servers"

[quote, README.md]
____
A curated list of awesome Model Context Protocol (MCP) servers.
____

This is a fork of link:https://github.com/punkpeye/awesome-mcp-servers[punkpeye/awesome-mcp-servers],
the community-maintained directory of MCP server implementations.
MCP is an open protocol enabling AI models to interact with local and remote
resources through standardised server implementations.

**Caveat:** This is a third-party fork. The upstream repository is the
canonical source. This fork exists for hyperpolymath-specific curation
and integration with link:https://github.com/hyperpolymath/boj-server[BoJ-server].
